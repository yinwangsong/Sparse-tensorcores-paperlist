# Awesome papers of sparsity in matrix processing units

Nowadays, the COTS AI accelerators are turning to the direction of integrating matrix processing units (include but not limited to tensor cores) into the chipset. Sparsity is the ever-lasting topic; here we curate the representive papers, FYI.

## NV Tensor Cores

\[EuroSys'25\] SpInfer: Leveraging Low-Level Sparsity for Efficient Large Language Model Inference on GPUs --- Bit map encoding for compressing the indices.

\[ASPLOS'24\] DTC-SpMM: Bridging the Gap in Accelerating GeneralSparse Matrix Multiplication with Tensor Cores

\[VLDB'24\] Flash-LLM: Enabling Cost-Effective and Highly-Efficient Large Generative Model Inference with Unstructured Sparsity --- CUDA Core Decompression, tensor core full compute
